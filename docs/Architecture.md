# 8-Bit CPU Architecture

Core Modules:

1. Program Counter (PC)
2. Memory Address Register (MAR)
3. 16×8 RAM
4. Instruction Register (IR)
5. Control Unit (FSM)
6. A Register
7. B Register
8. ALU
9. Flag Register
10. Output Register

Additional Modules:

1. Sensor Interface Module
2. Clock Divider Module
3. Status LED Module

Input Devices:

* 16 Slide Switches
* IR Sensor
* 4 Push Buttons

Output Devices:

* 7 Segment Display
* 16 LEDs

Clock Modes:

* 1 Hz
* 10 Hz

Processor Type:

Single Bus Architecture
